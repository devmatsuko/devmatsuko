[Magicians Society](https://magicians-society.com/)
「マジシャンのマジシャンによるマジシャンのためのSNS」をコンセプトに開発したSNSサイトです。  
マジックの演技動画の投稿や、自身が作成したトリック・製品の販売・購入などを行うサイトです。  
※本サイトはポートフォリオサイトとして作成したものであるため、実際に運用はしておりません。
